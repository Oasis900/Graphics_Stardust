{Features}
-

- Renders hardcoded Cubes and pyramid.

- Renders .obj loaded Torus Knot.

- Renders Skybox cube with Texture Cube applied, anchored to camera position.

- Debug Camera with full WASD functionality, alongside TFGH or arrow key camera view rotation.

- Blinn-Phong Shading, with diffuse, ambient and specular componenets applied per pixel.

---
{Controls}
-
- Q - Toggles Wireframe State
- E - Toggles Filled State
- 1 - Enable Static Camera Mode
- 2 - Enable Debug Camera Mode
- WASD - Forward, Backward, Left and Right Debug Camera Movement
- SPACE - Upward Debug Camera Movement
- SHIFT - Downward Debug Camera Movement
- TFGH or Arrow keys - Debug Camera eye rotation
